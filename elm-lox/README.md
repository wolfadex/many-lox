# elm-lox

An implementation of [jlox](https://craftinginterpreters.com/) in Elm.

## Development

#### Prerequisites

- [NodeJS](https://nodejs.org/en/)
- [Elm](https://elm-lang.org/)

```sh
npm install
npm run dev ../examples/scanner_complete.lox
```
